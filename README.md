# Package Installation (One by One)
### Cross-origin resource sharing

```sh
npm install cors
```

### Dotenv

```sh
npm install dotenv
```

### Express

```sh
npm install express
```

### Sequelize

```sh
npm install sequelize
```

### mysql2

```sh
npm install mysql2
```

### bcrypt

```sh
npm install bcrypt
```

### jsonwebtoken

```sh
npm install jsonwebtoken
```

### nodemailer

```sh
npm install nodemailer
```

### cloudinary

```sh

```

### multer
